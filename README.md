# Connect 5 (Gomoku)

A classic Connect 5 board game playable in the browser. Two players take turns placing black and white stones on a 15x15 grid. The first player to align 5 stones in a row — horizontally, vertically, or diagonally — wins.

## How to Play

1. Open `index.html` in a web browser.
2. Black goes first. Click an intersection on the board to place a stone.
3. Players alternate turns. Get 5 in a row to win.

## Features

- 15x15 board with traditional wooden appearance
- Turn indicator with stone preview on hover
- Last-move highlight and winning-line highlight
- Undo (button or Ctrl+Z)
- Score tracking across games
- Responsive layout that adapts to screen size
